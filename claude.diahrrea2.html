Global Latent Offset

Added to every dimension of every tile's own latent before decoding. Values are additive with animation. Use "Apply & Redraw" to push a one-shot render, or leave animation running for live effect.

Per-Tile Latent Offset

Each tile now evolves its own latent independently (seeded from its own encoded pixels, or the shared root + frozen jitter if no per-tile encoding is available). Effective latent = tile's own evolving latent + global offset + this per-tile offset.
